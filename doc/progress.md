# Antigravity Manager 开发进展文档

## 2026-01-14
### v1.4.0 更新
- **功能增强：账号导入导出**
    - 在账号列表标题栏增加“导入账号”和“导出所有账号”按钮。
    - 在单个账号上下文菜单增加“导出单个账号”功能。
    - 支持 JSON 格式，包含 Token 和指纹信息以便快速备份/同步。
- **功能增强：设备指纹管理**
    - 在账号详情节点显示已绑定的设备指纹摘要（machine_id 前8位）。
    - 优化“绑定设备指纹”交互，支持“采集当前设备指纹”和“生成全新随机指纹”两种模式。
- **功能优化**
    - 增加“删除账号”命令并带确认弹窗。
    - 优化侧边栏图标及菜单布局。
    - 更新版本号至 v1.4.0。
